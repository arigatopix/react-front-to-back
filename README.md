# GITHUB setting initial

`git init`

`touch .gitignore`

`git commit -m empty`

`git tag EMPTY`

## สร้าง project หรือ folder

`mkdir PROJECTNAME`

## สร้าง Branch หรือย้าย Branch

`git branch BRANCHNAME EMPTY`

`git checkout BRANCHNAME`

## เริ่มจากสร้าง Project อีกครั้ง

`git add .`

`git commit -m "..."`
